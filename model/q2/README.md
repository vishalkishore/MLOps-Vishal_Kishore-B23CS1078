# Q2 Models

This folder contains the Assignment 5 Q2 checkpoints required for upload.

- `q2_classifier_best_model.pt` - clean CIFAR-10 ResNet-18 classifier
- `q2_detector_pgd_best_model.pt` - ResNet-34 detector trained on PGD adversarial samples
- `q2_detector_bim_best_model.pt` - ResNet-34 detector trained on BIM adversarial samples
